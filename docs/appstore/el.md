## Description (μέγ. 4000 χαρακτήρες)

Μάθε λέξεις με τρόπο που έχει πλάκα — και πιάνει τόπο.

Το VocabSpark χρησιμοποιεί ένα δοκιμασμένο σύστημα καρτών (Spaced Repetition), ώστε οι λέξεις να μένουν για πολύ στη μνήμη σου. Οι κάρτες που δεν ξέρεις ακόμα εμφανίζονται πιο συχνά. Αυτές που τις έχεις μάθει, πιο σπάνια.

ΧΑΡΑΚΤΗΡΙΣΤΙΚΑ

• 16 γλώσσες — Γαλλικά, Αγγλικά, Ισπανικά, Ιταλικά και πολλές άλλες
• Φυσική προφορά — Κάθε λέξη ακούγεται με φυσική, καθαρή φωνή
• Αυτόματα παραδείγματα — Για κάθε λέξη μια απλή πρόταση μέσα σε πλαίσιο
• Βοήθεια μετάφρασης — Γράψε μια λέξη και μεταφράζεται αυτόματα
• Λειτουργία τεστ λεξιλογίου — Διάλεξε λέξεις και εξασκήσου μέχρι να τις μάθεις όλες
• Εισαγωγή CSV — Ολόκληρες λίστες λεξιλογίου με μία κίνηση
• Φίλτρο χρόνου — Εξάσκηση μόνο στις λέξεις της ημέρας, της εβδομάδας ή του μήνα
• Παρακολούθηση σερί — Δες το σερί εκμάθησής σου με μια ματιά
• Καθημερινή υπενθύμιση — Για να μην ξεχνάς καμία μέρα

ΠΩΣ ΛΕΙΤΟΥΡΓΕΙ

1. Διάλεξε γλώσσα και πρόσθεσε λέξεις
2. Ξεκίνα μια συνεδρία εξάσκησης — η εφαρμογή σού δείχνει μια λέξη
3. Αποκάλυψέ την και αξιολόγησε: «Ξανά», «Σχεδόν» ή «Το ήξερα!»
4. Το «Ξανά» ξαναέρχεται αμέσως, το «Σχεδόν» αργότερα, με το «Το ήξερα!» η λέξη θεωρείται μαθημένη

ΦΤΙΑΓΜΕΝΟ ΓΙΑ ΜΑΘΗΤΕΣ

Το VocabSpark είναι επίτηδες απλό. Καμία περίπλοκη ρύθμιση, καμία απόσπαση — μόνο εσύ και οι λέξεις σου. Ιδανικό για την προετοιμασία του επόμενου τεστ λεξιλογίου.

ΣΗΜΕΙΩΣΗ

Για την προφορά και τα παραδείγματα χρειάζεται ένα OpenAI API key (το ορίζεις στις ρυθμίσεις). Όλα τα δεδομένα μένουν στη συσκευή σου — χωρίς λογαριασμούς, χωρίς παρακολούθηση, χωρίς διαφημίσεις.
