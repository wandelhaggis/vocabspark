3. Drittanbieter: OpenAI

VocabSpark nutzt die OpenAI-API für drei Funktionen:

  • Aussprache (Text-to-Speech) – Einzelne Wörter oder kurze Sätze werden an OpenAI gesendet und als Audiodatei zurückgegeben. Die Audiodateien werden lokal gecacht.
  • Beispielsätze – Beim Anlegen einer Vokabel wird ein einfacher Beispielsatz von GPT-4o-mini generiert.
  • Übersetzung – Optionale automatische Übersetzung einzelner Wörter beim Hinzufügen.

Diese Aufrufe erfolgen direkt von deinem Gerät an die OpenAI-Server – wir sehen diese Daten nicht. Es gilt die Datenschutzerklärung von OpenAI. Du nutzt deinen eigenen API-Key, daher besteht ein direktes Vertragsverhältnis zwischen dir und OpenAI.

Alle lokalen Daten werden durch die Verschlüsselung von iOS geschützt (Data Protection). Die Kommunikation mit der OpenAI-API erfolgt ausschließlich über HTTPS.

3. Third Party: OpenAI

VocabSpark uses the OpenAI API for three features:

  • Pronunciation (Text-to-Speech) – Individual words or short sentences are sent to OpenAI and returned as audio files, which are cached locally.
  • Example sentences – When creating a vocabulary item, a simple example sentence is generated by GPT-4o-mini.
  • Translation – Optional automatic translation of individual words when adding vocabulary.

These calls are made directly from your device to OpenAI servers – we do not see this data. OpenAI's Privacy Policy applies. You use your own API key, establishing a direct relationship between you and OpenAI.

4. What We Do NOT Do

  • We operate no servers
  • We collect no personal data
  • We use no tracking, analytics, or cookies
  • We display no advertisements
  • We do not sell data to third parties
  • We create no user profiles

5. Children's Privacy

VocabSpark is designed for students. Since the app does not collect personal data, has no user accounts, and does not transmit data to our servers, it is suitable for use by minors. The OpenAI API key should be set up by a parent or guardian.

6. Data Security

All local data is protected by iOS Data Protection (device encryption). Communication with the OpenAI API uses HTTPS exclusively.

7. Your Rights

Since all data is stored locally on your device, you have full control. You can:

  • Delete all vocabulary at any time within the app
  • Remove the API key in Settings
  • Uninstall the app to permanently delete all data
  • Clear the TTS cache via iOS storage settings

8. Changes

If this privacy policy changes, the date at the top will be updated. For material changes, we will notify users via an app update.
