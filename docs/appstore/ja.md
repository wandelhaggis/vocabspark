## Description (max 4000 Zeichen)

楽しく続けられて、ちゃんと身につく単語学習を。

VocabSparkは、科学的に効果が実証されたフラッシュカード方式（間隔反復学習／Spaced Repetition）で、単語をしっかり長期記憶に定着させます。まだ覚えていないカードはくり返し、しっかり覚えたカードは間隔をあけて出題されます。

主な機能

• 16言語に対応 — フランス語、英語、スペイン語、イタリア語など
• 自然な発音 — すべての単語をはっきりとした自然な音声で再生
• 自動で例文を生成 — 各単語に、文脈のわかるやさしい例文をひとつ
• 翻訳サポート — 単語を入力するだけで自動翻訳
• 単語テストモード — 選んだ単語を、覚えるまでくり返し練習
• CSVインポート — 単語リストをまとめて一括登録
• 期間フィルター — 今日・今週・今月に追加した単語だけを練習
• 連続学習トラッカー — 学習ストリークをひと目で確認
• 毎日のリマインダー — 1日も忘れずに学習を続けられます

使い方

1. 言語を選んで、単語を登録します
2. 学習セッションを開始すると、単語がひとつずつ表示されます
3. 答えを表示したら、自己評価をタップ：「もう一度」「惜しい」「正解！」
4. 「もう一度」はすぐに再出題、「惜しい」は少し後に、「正解！」は完了です

学生のために

VocabSparkは、あえてシンプルに作られています。ややこしい設定も、気が散る要素もなし — あるのはあなたと単語だけ。次の単語テスト対策にぴったりです。

ご注意

発音と例文の生成にはOpenAIのAPIキーが必要です（「設定」画面で入力してください）。すべてのデータは端末内にのみ保存されます — アカウント登録なし、トラッキングなし、広告なし。

## What's New — v1.1.0 (max 4000 Zeichen)

• アプリの表示言語が16言語に対応：英語、ドイツ語、フランス語、スペイン語、イタリア語、ポルトガル語、オランダ語、スウェーデン語、ポーランド語、ロシア語、トルコ語、ギリシャ語、日本語、韓国語、中国語（簡体字）、アラビア語
• アラビア語では右から左へのレイアウトに対応
• 言語デッキがペア方式になりました：学習したい言語と自分の母語を作成時に選べるようになり、翻訳と読み上げもそれに合わせて切り替わります
• 初回起動時に、アプリの表示言語が端末のシステム言語に自動で合わせられます
• すべての画面テキストを16言語で校正しました
• 学習セッションを5回以上こなし、かつ前回から120日以上経っている場合に、一度だけApp Storeのレビューをお願いするダイアログが表示されます（うまくいかなかったセッションの直後にはスキップされます）
• 修正：方向選択の矢印絵文字が、アラビア語でも反転しないようになりました
• 修正：長い言語名（「オランダ語」「ポルトガル語」など）がグリッドのタイル内にきれいに収まるようになりました
