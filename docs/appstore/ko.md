## App Name (최대 30자)
VocabSpark: 나만의 단어장

## 부제 (최대 30자)
학교 공부용 단어 암기 앱

## 프로모션 텍스트 (최대 170자, 언제든 변경 가능)
16개 언어 단어를 자연스러운 발음, 예문, 똑똑한 복습 시스템으로 외워요. 중·고등학생에게 딱 맞아요.

## 설명 (최대 4000자)

재미있게, 그리고 확실하게 외우는 단어 학습.

VocabSpark는 검증된 플래시카드 방식(간격 반복 학습)으로 단어가 오래 기억에 남도록 도와줘요. 아직 잘 모르는 카드는 자주, 이미 익숙한 카드는 가끔씩만 나와요.

주요 기능

• 16개 언어 — 영어, 독일어, 스페인어, 프랑스어, 이탈리아어 등
• 자연스러운 발음 — 모든 단어를 또렷하고 자연스러운 목소리로 들어 봐요
• 자동 예문 — 단어마다 문맥 속 쉬운 예문이 함께 나와요
• 번역 도우미 — 단어를 입력하면 자동으로 번역해 줘요
• 단어 시험 모드 — 원하는 단어를 골라서 완벽하게 외울 때까지 연습해요
• CSV 가져오기 — 단어 목록을 한 번에 통째로 불러와요
• 기간 필터 — 오늘, 이번 주, 이번 달에 추가한 단어만 따로 연습해요
• 연속 학습 기록 — 며칠째 이어서 공부했는지 한눈에 확인해요
• 매일 알림 — 하루도 빠뜨리지 않게 알려 줘요

사용 방법

1. 언어를 고르고 단어를 입력해요
2. 학습을 시작하면 앱이 단어를 하나씩 보여 줘요
3. 뜻을 확인하고 스스로 평가해요: 「다시」, 「거의」, 「알아요!」
4. 「다시」는 바로 다시, 「거의」는 조금 뒤에, 「알아요!」는 끝이에요

학생을 위해 만들었어요

VocabSpark는 일부러 단순하게 만들었어요. 복잡한 설정도, 주의를 흐리는 요소도 없어요 — 오직 단어와 나뿐이에요. 다음 단어 시험을 준비하기에 딱 좋아요.

참고 사항

발음과 예문 기능을 쓰려면 OpenAI API 키가 필요해요(설정 화면에서 입력). 모든 데이터는 기기 안에만 저장돼요 — 계정도, 추적도, 광고도 없어요.

## 새 소식 — v1.1.0 (최대 4000자)

• 앱 인터페이스가 이제 16개 언어로 제공돼요: 영어, 독일어, 프랑스어, 스페인어, 이탈리아어, 포르투갈어, 네덜란드어, 스웨덴어, 폴란드어, 러시아어, 터키어, 그리스어, 일본어, 한국어, 중국어(간체), 아랍어
• 아랍어를 위한 오른쪽에서 왼쪽 레이아웃 지원
• 언어 덱이 이제 쌍으로 구성돼요: 덱을 만들 때 학습할 언어와 내 언어를 함께 지정하고, 번역과 TTS가 이에 맞춰 작동해요
• 처음 실행할 때 기기의 시스템 언어에 맞춰 인터페이스가 자동 설정돼요
• 모든 인터페이스 텍스트가 16개 언어 전부 검수 완료됐어요
• 학습 세션을 5번 이상 성공적으로 마치면(그리고 120일에 한 번 이하로) App Store 평가를 한 번 요청해요 — 세션 결과가 좋지 않으면 건너뛰어요
• 수정: 아랍어 모드에서 방향 선택 화살표 이모지가 더 이상 좌우 반전되지 않아요
• 수정: 긴 언어 이름(예: 「네덜란드어」, 「포르투갈어」)이 이제 격자 타일 안에 깔끔하게 들어가요
