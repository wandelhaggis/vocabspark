## 描述（最多 4000 字符）

背单词也可以很轻松——而且真的记得住。

VocabSpark 用的是经过验证的间隔重复算法（Spaced Repetition）：还没记牢的单词反复出现，已经会了的就少出现，让你把时间花在刀刃上。

主要功能

• 16 种语言 —— 法语、英语、西班牙语、意大利语等多种常用语言
• 地道发音 —— 每个单词都能听到清晰、自然的真人级发音
• 自动例句 —— 每个单词配一句简单的例句，在句子里记得更牢
• 翻译辅助 —— 输入一个词，自动帮你翻译
• 测验模式 —— 自选单词反复练，直到全部掌握
• CSV 导入 —— 一次性导入整份单词表
• 时间筛选 —— 只练今天、本周或本月新加入的单词
• 连续打卡 —— 一眼看到自己坚持了多少天
• 每日提醒 —— 不再漏掉任何一天

使用方法

1. 选择语言，输入单词
2. 开始学习，屏幕上会出现一个单词
3. 点一下显示答案，再自己评：「重来」「差一点」或「会了！」
4. 「重来」会马上再出现，「差一点」稍后再来，「会了！」这张就过了

为学生而做

VocabSpark 刻意做得简单。没有复杂设置，也没有多余干扰——就是你和你的单词。单元测验、默写前的复习利器。

说明

发音和例句功能需要填入 OpenAI API 密钥（在「设置」里）。所有数据都只留在本机——无需账号、不采集行为数据、没有广告。
