## Description (max 4000 Zeichen)

تعلّم المفردات بطريقة ممتعة — وفعّالة.

يعتمد VocabSpark على نظام البطاقات التعليمية المجرَّب (التكرار المتباعد) لترسيخ المفردات في الذاكرة على المدى الطويل. الكلمات التي لم تتقنها بعد تظهر لك بتكرار أكبر، والكلمات التي أتقنتها تظهر بتكرار أقل.

المزايا

• 16 لغة — الفرنسية، الإنجليزية، الإسبانية، الإيطالية، وغيرها الكثير
• نطق طبيعي — استمع إلى كل كلمة بصوت طبيعي وواضح
• جمل أمثلة تلقائية — جملة بسيطة لكل كلمة لفهمها في سياقها
• مساعد الترجمة — اكتب الكلمة ودع التطبيق يترجمها تلقائياً
• وضع اختبار المفردات — اختر الكلمات وتدرّب عليها حتى تتقنها
• استيراد CSV — استورد قوائم مفردات كاملة دفعة واحدة
• فلتر زمني — تدرّب على كلمات اليوم أو هذا الأسبوع أو هذا الشهر فقط
• تتبّع المواظبة — تابع سلسلة أيام تعلّمك المتواصلة
• تذكير يومي — حتى لا تفوّت أي يوم

طريقة الاستخدام

1. اختر اللغة وأدخل مفرداتك
2. ابدأ جلسة التعلّم — يعرض لك التطبيق كلمة
3. اكشف الإجابة ثم قيّم نفسك: «مرة أخرى»، «كدت!»، أو «أعرفها!»
4. «مرة أخرى» تعود فوراً، «كدت!» تعود لاحقاً، «أعرفها!» انتهت

مصمَّم للطلاب

صُمِّم VocabSpark ليكون بسيطاً عن قصد. لا إعدادات معقّدة ولا عناصر تشتّت الانتباه — أنت ومفرداتك فقط. مثالي للاستعداد لاختبار الكلمات القادم.

ملاحظة

يحتاج النطق وجمل الأمثلة إلى مفتاح OpenAI API (يُدخَل في الإعدادات). جميع البيانات تبقى على جهازك — بلا حسابات ولا تتبّع ولا إعلانات.
