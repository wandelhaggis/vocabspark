## Koreanisch (ko)

• 새 기능: 카드를 스와이프! 오른쪽 = "알아요!", 왼쪽 = "다시" (학습과 단어 시험 모두)

• 복습할 단어가 있을 때만 알림이 와요
• 단어 시험: 한 번의 스와이프로 여러 단어 선택
